# 团队Logo
***
## 设计理念
  在设计团队Logo时，我们围绕围绕核心意象、视觉符号、色彩体系三个方面来设计
  ### 1.核心主体
  我们选用小鹿作为主体，小鹿天生善于奔跑，对应着我们团队想要拥有的能**快速响应变化、高效执行任务**的核心能力的核心能力，同时,小鹿鹿角上的绿叶，象征着我们需要有能够持续成长，不断迭代的特质，能够不断向上突破，长期进步。
  ### 2.视觉符号
  这一部分主要定位在团队的功能与价值，小鹿的雪花围巾寓意着团队的**坚守与温度服务**，而贯穿画面的溪流寓意着**团队合作顺畅**，同时，背景的树林寓意团队应扎根行业，依托集体力量成长
  ### 3.色彩体系
  整图采用暖橙色为主要色调，橙色自带活力，温暖和积极的属性，彰显出团队的**热情与创造力**，而背景的绿色，对应团队**稳健发展**的理念

  ## 生成过程
  我们使用了GORK4.1的模型生成图片，输入了积极，鹿，卡通这三个关键词
